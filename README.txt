注： 设置有默认和自己构造两个模式 默认 输入1则会按照已经设定好的数据自行构造 
        使用自己构造模式请按照下述说明

若选择自己构造（选择0） 
输入包括 ：1.底面点的坐标 请按照顺序输入 外环 内环（按照逆时针方向输入）
           2.扫掠方向（单位向量）以及扫掠方向上的边长。
例：
{
    （0,0,0）；
   （10,0,0,）；
    （10,10,0）；
   （0,10,0,）；
    （1,1,0）；
   （9,1,0,）；
    （1,4,0）；
   （1,6,0,）；
    （9,6,0）；
   （1,9,0,）；
}
扫掠方向 （0,0,1） 边长 2

输入上的一点要求 ：底面的10个点（4,3,3），设置为一个矩形内挖两个三角形，将矩形沿y轴方向对半分开，两个三角形 应当分别在两个矩形中。

原因：对于存在两个洞的一个面，很难有一个鲁棒性很好的策略能够涉及到各类特殊情况，能想到的一个解决方法
是沿着某个洞的一个边将矩形划分成两部分，在每个部分做一次带一个洞的面的绘制，这里的将底面拆分就是这个思想的简化版。
